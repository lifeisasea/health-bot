# Health bot (Этап 1)

Телеграм-бот: дневник питания по фото + советы по запросу. Работает на MiniMax-M3.
(Шапка выше — конфиг для Hugging Face Spaces, на работу бота не влияет.)

## Что уже умеет
- 📷 Фото еды → оценка калорий и БЖУ, запись в дневник.
- 💬 Ответы на вопросы («что съесть на обед?», «полезно ли это?», «чем заняться сегодня?») с учётом аллергий, цели и состояния здоровья.
- 🌙 Каждый вечер — разбор питания за день.
- 📊 Раз в неделю — сводка и рекомендации (учитывает периоды болезни/ЭКО).
- 🩺 Состояния: аллергии (постоянно), болезнь/ЭКО/травма (временно, включаются и выключаются командой).

Анализы (фото/PDF) и данные с часов Garmin — следующие этапы.

**Модели:** основная — MiniMax-M3 (дёшево). Claude подключается резервом, когда MiniMax
не справился (не разобрал блюдо, сработала модерация, ошибка). Резерв включается сам,
если в `.env` задан `ANTHROPIC_API_KEY`; иначе бот работает только на MiniMax.

## Запуск (один раз настроить)

1. Установить Python 3.10+.
2. В папке проекта:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
3. Создать бота: в Telegram написать **@BotFather** → `/newbot` → получить **токен**.
4. Узнать свой ID: написать **@userinfobot** → он пришлёт число.
5. Открыть файл `.env` и вписать `TELEGRAM_BOT_TOKEN` и `OWNER_ID` (ключ MiniMax уже вписан). По желанию — `ANTHROPIC_API_KEY` для резерва Claude.
6. Запустить:
   ```bash
   python bot.py
   ```
7. В Telegram открыть своего бота, нажать **Start**.

Пока скрипт запущен на компьютере — бот работает только при включённом ноутбуке.

## Деплой на Hugging Face Spaces (24/7, бесплатно)

1. На huggingface.co → **New Space** → SDK **Docker**, видимость **Private**.
2. В Space → **Settings → Variables and secrets** добавить секреты:
   - обязательные: `MINIMAX_API_KEY`, `TELEGRAM_BOT_TOKEN`, `OWNER_ID`;
   - резерв Claude: `ANTHROPIC_API_KEY`;
   - бэкап истории: `HF_TOKEN` (токен с правом записи), `HF_BACKUP_REPO` (напр. `ЛОГИН/health-bot-data`).
3. Залить код в репозиторий Space:
   ```bash
   git init && git add -A && git commit -m "health bot"
   git remote add space https://huggingface.co/spaces/ЛОГИН/НАЗВАНИЕ
   git push space main
   ```
4. Space сам соберёт Docker и запустит бота. Файл `.env` НЕ заливается (секреты берутся из настроек Space).

История (дневник, аллергии, состояния) автоматически бэкапится в приватный датасет и
восстанавливается при перезапусках — данные не теряются.
